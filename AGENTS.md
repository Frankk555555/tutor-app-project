# Agent Context

## Design Context
This project uses the Impeccable frontend design framework to maintain visual consistency and alignment with strategic product goals.

- **Strategic Guidelines**: Refer to [PRODUCT.md](file:///c:/PROJECT/Tutor-Platform2/PRODUCT.md) for registers, target audience, brand personality, and core design principles.
- **Visual Guidelines**: Refer to [DESIGN.md](file:///c:/PROJECT/Tutor-Platform2/DESIGN.md) for details on colors, typography, elevation, components, and specific Do's and Don'ts.
- **Design Configuration**: Configured under [.impeccable/design.json](file:///c:/PROJECT/Tutor-Platform2/.impeccable/design.json).
- **Core Principles**:
  - **Clarity Over Clutter**: Ensure spacing is comfortable and layouts are structured for Thai students and tutors.
  - **Supportive Trust**: Showcase credentials and schedules clearly to build confidence.
  - **Inclusive Legibility**: All Thai typography must maintain a minimum line-height of 1.5.
